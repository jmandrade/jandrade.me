About this site

This is a "serverless" static site. I wrote the code using Bootstrap 4 and JQuery, and it's being hosted on an GCP Google Cloud Storage bucket. To send the information provided when using the Contact form I use AWS API Gateway and AWS Lambda. There's no server for me to support (including updates and troubleshooting). The content is provided securely (encrypted using TLS over HTTP) by using a GCP Global Load Balancer.
