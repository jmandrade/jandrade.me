About this site

This is a "serverless" site. I wrote the code using Bootstrap and JQuery, and it's being hosted on an AWS S3 bucket. To send the information provided when using the Contact form I use AWS API Gateway and AWS Lambda. There's no server for me to support (including updates and troubleshooting). The content is provided securely (encrypted using TLS over HTTP) by using AWS CloudFront.
